PORTAESCALERA MODELO GRAN ECO

Modelo Gran Eco para traslado de escaleras extensibles rectas de dos tramos, con tres filas de rodillos y trabaescalera con portacandado. Largo 2 mts
❮ ❯

PORTAESCALERA MODELO PARRILLA

Modelo Parrilla para traslado de escaleras extensibles rectas de dos tramos, con rodillos traseros y trabaescalera con portacandado, además cuenta con 50 cm libre para traslado adicional según lo que el cliente necesite. Largo estándar 1.5 mts.
❮ ❯

PORTAESCALERA MODELO DOBLE

Modelo Doble para traslado de dos escaleras extensibles rectas de dos tramos, con rodillos traseros y trabaescalera con portacandado. Largo estándar 2 mts.
❮ ❯

PORTAESCALERA CON CAÑERA

Modelo con Cañera para traslado de una escalera extensible rectas de dos tramos, con rodillos traseros y trabaescalera con portacandado. Además cuenta con la posiblilidad de traslado de caños de 2 mts y 3 mts de largo.
